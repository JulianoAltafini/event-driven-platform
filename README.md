# Event Driven Platform

Demo de uma plataforma orientada a eventos utilizando arquitetura baseada em serviços AWS (simulados com LocalStack).

## Arquitetura
- Frontend: HTML + JavaScript
- Backend: API REST
- Mensageria: SQS
- Processamento assíncrono: Lambda
- Storage: S3 / DynamoDB
- Observabilidade: CloudWatch

## Objetivo
Demonstrar conceitos de arquitetura moderna, desacoplamento, escalabilidade e boas práticas de engenharia de software.
